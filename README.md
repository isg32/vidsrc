## Install

```sh
pip install imdbpy

pip install webbrowser

git clone https://github.com/isg32/vidsrc/

python3 vidsrc.py
```

## Bug Fixes
```
python3 -m pip install --user imdbpy

python3 -m pip install --user webbrowser

```
